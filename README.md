# Compiler Design Lab Test

A modern, glassmorphic React application containing practical examples and lab experiments for Compiler Design, including Lex/Flex, Yacc/Bison, and C/C++ implementations.

## Experiments Included:
- **Exp 3**: Lexical Analyzer using FLEX
- **Exp 4**: Capital Letter Count using FLEX
- **Exp 5**: Elimination of Left Recursion (C/C++)
- **Exp 6**: Calculator using FLEX + BISON
- **Exp 7**: FIRST & FOLLOW Sets for Top-Down Parsing (C/C++)
- **Exp 8**: Predictive Parsing Table (C/C++)

## Features:
- Split-pane layout for seamlessly viewing code and execution instructions/terminal commands side-by-side.
- Multi-language support with an inline toggle (e.g. swap smoothly between C++ and C code representations).
- Premium dark mode aesthetics with responsive glassmorphic cards and hover effects.
- One-click "Copy to Clipboard" with a custom success toast notification.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the application.
